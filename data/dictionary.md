**Légende des colonnes**  
- **Variable** : nom exact de la variable dans le fichier de données  
- **Description** : signification, contexte ou unité  
- **Type** : numérique, chaîne, date, booléen, catégoriel, etc.  
- **Modalités / Valeurs possibles** : pour les variables catégorielles ou à valeurs codées  
- **Valeurs manquantes** : code utilisé pour les valeurs manquantes (ex : NA, -1)  
- **Source / Fichier** : nom du fichier ou de la table d’origine  
- **Remarques** : toute information utile (ex : transformation, anonymisation)
